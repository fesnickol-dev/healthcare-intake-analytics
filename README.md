Healthcare Patient Intake & Eligibility Performance Dashboard
Overview
This project analyzes simulated healthcare intake data to evaluate operational efficiency, eligibility approval rates, and documentation accuracy.
Tools Used
SQL
Excel
Tableau
Key KPIs
Total Registrations
Approval Rate
Average Processing Time
Documentation Error Rate
Key Insights
Medicaid cases showed longer processing times.
Evening shifts demonstrated higher documentation error rates.
Self-Pay patients had higher denial percentages.
Business Recommendations
Improve documentation review process during high-volume shifts.
Optimize Medicaid processing workflow.
Allocate staff resources during peak intake months.
